## :icon{name="i-noto-warning"} Breaking Features

### Nuxt v3.16

To avoid hoisting issues with the new Unhead v2, the Nuxt OG Image v5 requires Nuxt v3.16.

Please upgrade your Nuxt to continue using OG Image.

```bash
nuxi upgrade --force
```
